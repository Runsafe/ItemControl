Item Control
============

Our shop feature lives in here, among other things

/traders info

permission: runsafe.traders.info

Run this command and right click a shop button, it will give you information about it.

/traders delete

permission: runsafe.traders.delete

Run this command and right click a shop button, it will delete it **without** removing tag data.

/traders createshop <compare_name> <compare_durability> <compare_lore> <compare_enchants>

permission: runsafe.traders.create

Run this command and right click a button. it will turn it into a shop, or if it's already a shop it will overwrite it with the new values and give you a chance to change its contents.
When it opens an inventory, items on the top row are the payment, items on the bottom row are the product.
